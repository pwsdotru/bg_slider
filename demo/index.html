BG Slider demo

The bg slider is plugin for jQuery. It allow slide images on background for any container with any data: text, images, forms and etc.

For this plugin you should use jQuery. Plugin tested with jQuery 1.x

Plugin allow use custom options:

  • timeout - time in seconds for show each slide;
  • speed - time in seconds for switch slides.
For details see example 2.

Example 1 - Default options

1. Define container and set list of images for slider in attribute data-sliders:
<div data-sliders="sea/img1.jpg;sea/img2.jpg;sea/img3.jpg;sea/img4.jpg" id="demosea"> ... </div>

2. Run slider with default options:
$("#demosea").bg_slider();

Example 2 - Custom options

1. Define container and set list of images for slider in attribute data-sliders:
<div data-sliders="fruits/img1.jpg;fruits/img2.jpg;fruits/img3.jpg;fruits/img4.jpg" id="demofruit"> ... </div>

2. Run slider with custom options:
$("#demofruit").bg_slider({timeout:10, speed:3});

Options: timeout - time in seconds for show each slide, speed - time in seconds for switch slides.
